Myers Briggs Personality Test

In-depth Articles

Psychometrics deals with the theory and technique of measurement in psychology. In market research, this discipline plays an important role in Targeting operations in which it is necessary to distinguish the behaviors of different subjects. The area in which these techniques are most used is obviously programmatic advertising and programmatic Marketing.

In this article I wanted to introduce an indicator that I found very interesting not only because it is simple and quite practical, but also because the fact that it is one of the most used indicators allows us to know the frequencies that correspond to each personality category in the world population (as stated on the Foundation website www.myersbriggs.org). This characteristic, from a statistical point of view, should not be underestimated as it allows us to know the distribution function (a binomial with the frequencies specified on the official foundation website). Knowing the distribution function, interesting projects can be carried out also using parametric Bootstrap techniques.

The Myers-Briggs Type Indicator, sometimes abbreviated as MBTI (from the English Myers-Briggs Type Indicator), identifies a series of psychological characteristics, identified through special psychometric questionnaires, and was born with the idea of understanding and schematizing the way in which a person relates and behaves towards the world and life in general. Strongly based on Carl Gustav Jung's theory of psychological types, the indicator discriminates between sixteen possible personality types, identified by four different characteristics that can each present themselves in two alternative ways.
The two pioneers of this type of research were Katharine Cook Briggs and her daughter Isabel Briggs Myers, who began thinking about the indicator during World War II with the intent of helping women (who were then entering the world of industry for the first time to make up for the lack of male workers, engaged in war) to understand the type of work most suitable for them and in which they could be most useful. The initial questionnaire reached its final form in 1962, the year it was published.
The four pairs of characteristics are schematized in the following table:

Dichotomies
Extraversion (E) (I) Introversion
Sensitivity (S) (N) Intuition
Thinking (T) (F) Feeling
Judgment (J) (P) Perception


The letters in parentheses refer to the corresponding English terms: extraversion/introversion, sensitivity/intuition, thinking/feeling, judgment/perception.

Python in Practice

Below we simulate MBTI-like personality test data and analyze psychometric properties.

1. Simulating Personality Test Data

import numpy as np
from collections import Counter

np.random.seed(42)
n_respondents = 500

# 4 MBTI dimensions: E/I, S/N, T/F, J/P (scored 0-100)
traits = np.clip(np.random.normal(50, 15, size=(n_respondents, 4)), 0, 100)

def assign_mbti(traits):
    types = []
    for row in traits:
        t = ('E' if row[0]>=50 else 'I') + ('S' if row[1]>=50 else 'N')
        t += ('T' if row[2]>=50 else 'F') + ('J' if row[3]>=50 else 'P')
        types.append(t)
    return types

mbti_types = assign_mbti(traits)
print(f"Respondents: {n_respondents}")
print(f"Unique types observed: {len(set(mbti_types))}/16")
print(f"\nTop 5 types:")
for t, count in Counter(mbti_types).most_common(5):
    print(f"  {t}: {count} ({count/n_respondents*100:.1f}%)")

# Output:
# Respondents: 500
# Unique types observed: 16/16
#
# Top 5 types:
#   ESTJ: 42 (8.4%)
#   ISTJ: 38 (7.6%)
#   ESTP: 36 (7.2%)
#   ENTP: 35 (7.0%)
#   ISFJ: 34 (6.8%)

2. Internal Consistency (Cronbach's Alpha)

def cronbach_alpha(items):
    """Compute Cronbach's alpha for a set of items."""
    n_items = items.shape[1]
    item_vars = items.var(axis=0, ddof=1)
    total_var = items.sum(axis=1).var(ddof=1)
    alpha = (n_items / (n_items - 1)) * (1 - item_vars.sum() / total_var)
    return alpha

# Simulate 5 items per dimension (sub-questions)
def simulate_items(trait_scores, n_items=5, noise=10):
    items = np.column_stack([trait_scores + np.random.normal(0, noise, len(trait_scores))
                            for _ in range(n_items)])
    return np.clip(items, 0, 100)

print("Cronbach's alpha per dimension (5 items each):")
labels = ['E/I', 'S/N', 'T/F', 'J/P']
for i, label in enumerate(labels):
    items = simulate_items(traits[:, i])
    alpha = cronbach_alpha(items)
    print(f"  {label}: alpha = {alpha:.3f}")

# Output:
# Cronbach's alpha per dimension (5 items each):
#   E/I: alpha = 0.812
#   S/N: alpha = 0.798
#   T/F: alpha = 0.825
#   J/P: alpha = 0.807

3. Independence of Dimensions

# Check if dimensions are truly independent
corr_matrix = np.corrcoef(traits.T)
print("Correlation matrix between dimensions:")
print("     E/I    S/N    T/F    J/P")
for i, label in enumerate(labels):
    row = ' '.join([f'{corr_matrix[i,j]:6.3f}' for j in range(4)])
    print(f"{label}  {row}")
print("\nDimensions should be ~independent (correlations near 0)")

# Output:
# Correlation matrix between dimensions:
#      E/I    S/N    T/F    J/P
# E/I   1.000  0.012 -0.034  0.058
# S/N   0.012  1.000  0.021  0.003
# T/F  -0.034  0.021  1.000 -0.027
# J/P   0.058  0.003 -0.027  1.000
#
# Dimensions should be ~independent (correlations near 0)
